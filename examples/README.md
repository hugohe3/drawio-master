# 示例文件

本目录包含各种类型的 Draw.io 图表示例。

## 示例列表

### 1. claude_workflow.drawio
**类型：** 流程图  
**描述：** Claude AI 对话工作流程  
**适用场景：** AI 对话流程、决策树

### 2. claude_code_workflow.drawio
**类型：** 流程图  
**描述：** Claude 代码生成和执行工作流  
**适用场景：** 代码生成、工具执行流程

### 3. cooking_at_home.drawio
**类型：** 流程图  
**描述：** 在家烹饪的完整流程  
**适用场景：** 生活流程、操作指南

### 4. government_investment_workflow.drawio
**类型：** 泳道图  
**描述：** 政府投资审批流程  
**适用场景：** 政务流程、跨部门协作

### 5. supermarket_layout.drawio
**类型：** 布局图  
**描述：** 超市布局设计  
**适用场景：** 空间布局、平面设计

## 如何使用

1. **查看学习**：在 [Draw.io](https://app.diagrams.net/) 中打开
2. **作为模板**：另存为新文件后修改内容
3. **学习描述**：参考示例学习如何描述需求

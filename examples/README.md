# 示例文件

本目录包含 13 个精美的 Draw.io 图表示例，涵盖所有常用图表类型。每个示例都包含真实业务场景，可直接在 [Draw.io](https://app.diagrams.net/) 中打开使用。

---

## 📊 示例列表

### 1. flowchart_onboarding.drawio
**类型：** 流程图  
**描述：** 员工入职完整流程  
**适用场景：** HR流程、业务流程、决策树  
**特点：** 
- ✅ 包含开始/结束节点
- ✅ 菱形决策节点
- ✅ 循环流程（审核不通过返回）
- ✅ 统一配色方案

**核心流程：** 开始 → 提交申请 → HR审核 → 培训 → 分配设备 → 开通权限 → 完成

## 💡 如何使用

### 方式一：作为学习材料
1. 在 [Draw.io](https://app.diagrams.net/) 中打开示例文件
2. 查看图表结构和布局设计
3. 学习节点样式和连接方式
4. 理解不同图表类型的特点

### 方式二：作为模板使用
1. 选择合适的示例文件
2. 在 Draw.io 中打开
3. 另存为新文件
4. 修改内容为自己的业务场景
5. 调整样式和布局

### 方式三：学习需求描述
通过示例文件学习如何向 AI 描述图表需求：
- 明确图表类型
- 列出所有节点和关系
- 说明特殊要求（颜色、布局等）

# 贡献指南

感谢您对本项目的关注！欢迎任何形式的贡献。

## 如何贡献

### 报告问题

如果发现 bug 或有建议：
1. 查看 [Issues](../../issues) 确认是否已被报告
2. 创建新 Issue，描述问题和重现步骤

### 提交代码

1. Fork 项目
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m "feat: 描述"`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

## 提交规范

- `feat:` - 新功能
- `fix:` - Bug 修复
- `docs:` - 文档更新
- `style:` - 代码格式调整

## 贡献方向

- 优化提示词（`roles/drawio_prompt.md`）
- 添加示例文件（`examples/`）
- 完善文档
- 创建图表模板

## 行为准则

- 使用友好和包容的语言
- 尊重不同的观点
- 接受建设性批评
